
# Django

创建一个 Django 项目 `djangoadmin startproject <name> [directory]`

```bash
.
├── 1. 目录介绍
├── manage.py   # 管理文件，对项目进行操作
└── mysite
    ├── __init__.py
    ├── settings.py     # 配置文件
    ├── urls.py         # 路由 --> URL 和函数的对应关系
    └── wsgi.py         # 用来运行 web server

1 directory, 6 files
```

运行一个项目，默认是本地 8000 端口，可以指定 `python manage.py runserver 0.0.0.0:80`
